## Analytics ETL Pipelines

Collection of data pipelines designed to extract, transform, and prepare
data for analytical and reporting use cases.

These ETL workflows integrate multiple data sources, apply data cleansing
and standardization rules, and generate trusted analytical datasets used
across dashboards and operational reports.

All data structures, names, and examples have been anonymized for portfolio
presentation purposes.

### Key responsibilities
- Integration of multiple raw data sources
- Data cleansing and normalization
- Business rule application and enrichment
- Creation of analytical base tables
- Pipeline orchestration and scheduling
- Monitoring of execution status and failures

**Tools:** SQL, ETL orchestration tools, Data modeling
