## Client Profile Dashboard

Analytical dashboard designed to understand customer demographics, behavior,
and portfolio penetration.

This dashboard supports strategic decisions related to customer segmentation,
product offering, and lifecycle management.

All data displayed has been anonymized and adjusted for portfolio
presentation purposes.

### Key analyses
- Customer demographic segmentation
- Product and investment thesis penetration
- Portfolio concentration patterns
- Customer lifecycle timing and progression
