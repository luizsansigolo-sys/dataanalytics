## Daily Operational Monitoring Report

Operational report designed for daily monitoring of capital raising
performance and execution.

This report supports short-term decision-making by tracking progress
against targets, identifying bottlenecks, and highlighting opportunities
across channels, originators, and products.

All data displayed has been anonymized and adjusted for portfolio
presentation purposes.

### Key analyses
- Daily, weekly, and monthly capital raised
- Performance versus targets
- Distribution by channel (B2B / B2C)
- Distribution by originator and product
- Shelf status and open operations
- Week-over-week performance

**Tools:** Tableau, SQL
